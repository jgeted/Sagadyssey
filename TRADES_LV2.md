### 随机池

从全职业 Lv2 汇总池（66 项）中随机抽取 **3 项**。加价 **20-50%**：

- 原出售项 → 商人出售 cost ×（1 + 加价率），向下取整，最低 1
- 原收购项 → 商人收购 cost ×（1 + 加价率），向上取整

**以下 7 项从商人池移除**（单件 2 绿宝石定价，无法打折获利）：

## Lv1 → Lv2 设计思路

Lv2 各职业的变化方向：

| 职业 | Lv1 定位 | Lv2 定位 | 关键变化 |
|------|---------|---------|---------|
| 铁匠 | 石质时代 | 铁质时代 | 全铁工具 + 盾牌，取消材料交易 |
| 农民 | 生食材 | 加工食物 | 面包/南瓜派/曲奇，取消基础作物回收 |
| 工人 | 基础建材 | 加工建材 | 砖/玻璃 + 铁质工具，新增燧石/线回收 |
| 战士 | 石剑皮革 | 铁器锁链 | 铁武 + 铁盔靴 + 锁链胸甲，新增火药回收 |
| 弓手 | 弓 + 木箭 | 弩 + 附魔 | 弩 + 力量I书 + 光灵箭，新增燧石回收 |
| 重甲兵 | 锁链甲 | 铁甲 | 全铁甲 + 钻石剑（单件高价值），新增黑曜石回收 |
| 医师 | 不收不卖 | 药水入门 | 治疗/再生药水 + 金苹果，收购下界材料 |
| 诗人 | 书卷材料 | 附魔音乐 | 附魔书 + 唱片机 + 音符盒，新增青金石/皮革回收 |
