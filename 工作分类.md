## 六、优先级汇总

```
必须最先做（阻塞后续）：
  0.1 环境搭建 → 0.2 微型模组 → 0.3 配置文件 → 0.4 网络通信 → 1.1 研究点数

然后按模块顺序推进：
  Core 剩余 → NPC A 组 → NPC B 组 → NPC C 组 → NPC D 组 → Structure → Vehicle
```

**标"硬"的任务（共 7 个）：**
- 1.4 技能树 GUI
- 2.1 NPC 实体基类
- 2.9 工人工作 AI
- 2.10 战斗 AI
- 3.3 半透明预览
- 4.1 马/驴寻路改进

这 7 个是技术高点——不会也别慌，Claude Code 帮你攻克，你负责理解。
